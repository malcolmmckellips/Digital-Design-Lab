void cmu_open ( void  )

Enables all necessary oscillators and routes the clock tree for the application.

This function will enable the HFPER clock and disable the LFRCO oscillator and LFXO. It will also route the low frequency rc oscillator to the correct low frequency clock tree which will then be enabled.

Note
As this function is void, it will always format the same clock tree. To modify the clock tree, the enums within the function must be changed.
