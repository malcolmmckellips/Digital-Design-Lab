◆ current_block_energy_mode()

uint32_t current_block_energy_mode ( void  )

Returns the index of which energy mode is currently blocked by periphals.

Will search through the lowest_energy_mode array starting from EM0 until a blocked energy mode is found.

Note
If no blocked energy mode is found, the device will return the lowest possible energy mode.
Returns
A 32 bit unsigned integer corresponding to the current blocked energy mode.

◆ enter_sleep()

void enter_sleep ( void  )

Used to enter a sleep mode based on which energy modes are blocked by peripherals.

Puts the device in the deepest sleep currently allowed by peripherals. Will check sleep modes starting from EM0 until it finds one that is blocked.

Note
If no blocked sleep mode is found, the device will enter EM3.

◆ lowest_energy_mode

int lowest_energy_mode[MAX_ENERGY_MODES]
static

Sleep Routine Management Module.

This module is based on Silicon Labs sleep.c file and contains functions for the support of easy transition between and blocking of sleep modes. The user can use this file to enter the lowest sleep mode possible that is not blocked by peripherals.
